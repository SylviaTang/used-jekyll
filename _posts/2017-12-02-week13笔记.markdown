## Ruby安装与应用
### 安装
1. 点击安装老师给的下载包：rubyinstaller-2.4.2-2-*64.exe
2. 按顺序依次1、2、3，分别安装
3. 完成安装后，点击键盘：win+R，
4. 输入PowerShell并回车打开
5. 在Powershell界面输入“gem install bundle”
6. 输入 gem install jekyll
